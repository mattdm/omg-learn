# omg-learn

This project provides the `omg-learn` skill for learning from mistakes and creating preventive patterns.

## Skill Discovery

**Claude Code:** Skills are auto-discovered from `~/.claude/skills/omg-learn/`

**Cursor:** Use the Cursor rule in `~/.cursor/rules/omg-learn.mdc` (points to `~/.cursor/skills/omg-learn/SKILL.md`)

## Installation

See [README.md](README.md) for platform-specific installation instructions.

## Dependencies

The `omg-learn` skill depends on the `skill-creator` skill for creating new skills.
