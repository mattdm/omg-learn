# Changelog

All notable changes to the OMG! Learning Skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-02

### Added
- Initial release of the OMG! Learning Skill
- Core workflow for learning from user corrections triggered by "omg!"
- Support for both global and project-local skills
- Integration with skill-creator workflow
- Automatic skill registration via openskills sync
- Complete documentation and examples
- Public domain license (Unlicense)

### Features
- Detects "omg!" expressions during corrections
- Analyzes corrections to extract learnable knowledge
- Creates new skills or updates existing ones
- Handles global vs project-local skill placement
- Embeds essential skill structure guidance
- Progressive disclosure principle support
